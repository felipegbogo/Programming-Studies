Introdução ao Desenvolvimento Web

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas web. Ele é responsável por estruturar o conteúdo da página, como textos, imagens, links, entre outros elementos.

Estrutura básica de um documento HTML

Um documento HTML é composto por uma estrutura básica que inclui as seguintes tags:

  • <!DOCTYPE html>: Declaração do tipo de documento.
  • <html>: Tag que indica o início do documento HTML.
  • <head>: Tag que contém informações sobre o documento, como o título e links para arquivos CSS ou JavaScript.
  • <body>: Tag que contém o conteúdo visível da página.

Observação: O atributo rel é usado para especificar a relação entre o documento atual e o documento referenciado. Por exemplo, rel="stylesheet" indica que o link aponta para um arquivo de estilo CSS.

Exemplo de código HTML

A seguir, um exemplo de código HTML que utiliza as tags mencionadas:

<!DOCTYPE html>
<html>
    <head>
        <title>Meu Primeiro Documento HTML</title>
    </head>
    <body>
        <h1>Olá, mundo!</h1>
        <p>Este é um parágrafo.</p>
    </body>
</html>

Este código cria uma página HTML simples com um título e um parágrafo de texto.

As tags HTML são elementos que definem a estrutura e o conteúdo de uma página web. Cada tag é composta por um nome entre colchetes angulares, como <h1> ou <p>. As tags podem ser abertas e fechadas, como <h1> e </h1>, ou podem ser de auto-fechamento, como <br>.

Corpo de texto e formatação

Dentro do corpo de um documento HTML, podemos adicionar diversos elementos para formatar o texto e organizar o conteúdo. Alguns exemplos de formatação incluem:

  • <h1> a <h6>: Tags de título, onde <h1> é o título mais importante e <h6> é o menos importante.
  • <strong>: Tag para destacar texto em negrito.
  • <em>: Tag para destacar texto em itálico.
  • <p>: Tag para criar parágrafos de texto.
  • <br>: Tag para inserir uma quebra de linha.
  • <li>: Tag para criar itens de uma lista.
  • <pre>: Tag para formatar texto em predefinição (monospace).
    • Isto é, o texto dentro da tag <pre> será exibido com espaçamento e formatação preservados, o que é útil para exibir código ou texto que requer uma formatação específica.
  • <ul>: Tag para criar listas não ordenadas.
  • <ol>: Tag para criar listas ordenadas.
Observação: <strong> e <em> são tags de formatação semântica, ou seja, elas indicam o significado do conteúdo em vez de apenas o estilo visual.
Observação: <br> é uma tag de auto-fechamento que insere uma quebra de linha.

Com esses elementos, podemos criar páginas HTML bem estruturadas e formatadas. Por exemplo, 

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<strong>Texto em negrito</strong>
<em>Texto em itálico</em>
<p>Este é um parágrafo de texto.</p>
<br>
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Este código cria uma página HTML com um título principal, um subtítulo, texto em negrito, texto em itálico, um parágrafo de texto, uma quebra de linha e uma lista de itens.

Botões

Para criar um botão em HTML, utilizamos a tag <button>. O conteúdo dentro da tag <button> será o texto exibido no botão. Por exemplo:

<button>Clique aqui</button>

Este código cria um botão com o texto "Clique aqui".

Podemos também adicionar atributos ao botão para personalizá-lo, como o atributo type para definir o tipo do botão (por exemplo, "submit" para enviar um formulário) ou o atributo onclick para adicionar uma ação quando o botão for clicado. Por exemplo:

<button type="submit" onclick="alert('Botão clicado!')>Enviar</button>

Este código cria um botão do tipo "submit" que exibe um alerta com a mensagem "Botão clicado!" quando é clicado. Podemos também utilizar o ambiente <Link> para criar links em botões, mas isso é mais comum em linguagens de programação como JavaScript ou CSS. Em HTML puro, os botões são usados principalmente para interações no próprio documento.

Imagens

Para inserir uma imagem em uma página HTML, utilizamos a tag <img>. A tag <img> é uma tag de auto-fechamento, o que significa que não precisa de uma tag de fechamento. Para inserir uma imagem, precisamos usar o atributo src para especificar o caminho da imagem e o atributo alt para fornecer uma descrição alternativa da imagem. Por exemplo:

<img src="caminho_da_imagem.jpg" alt="Descrição da imagem" />

Este código insere uma imagem com o caminho "caminho_da_imagem.jpg" e a descrição "Descrição da imagem". Podemos também ajustar o tamanho da imagem utilizando os atributos width e height. Por exemplo:

<img src="caminho_da_imagem.jpg" alt="Descrição da imagem" width="200" height="150" />

Este código insere uma imagem com o caminho "caminho_da_imagem.jpg", a descrição "Descrição da imagem" e ajusta o tamanho da imagem para 200 pixels de largura e 150 pixels de altura.

Links

Para criar um link em HTML, utilizamos a tag <a> com o atributo href para especificar o destino do link. O conteúdo dentro da tag <a> será o texto exibido como link. Por exemplo:

<a href="https://www.exemplo.com">Clique aqui para visitar o site</a>

Este código cria um link com o texto "Clique aqui para visitar o site" que direciona para "https://www.exemplo.com" quando clicado. Podemos também criar links para outras páginas dentro do mesmo site utilizando caminhos relativos. Por exemplo:

<a href="pagina2.html">Ir para a Página 2</a>

Este código cria um link com o texto "Ir para a Página 2" que direciona para "pagina2.html" quando clicado, assumindo que "pagina2.html" esteja no mesmo diretório do arquivo HTML atual.

Questões Importantes

  1. Por qual motivo o comando <img> possui uma barra (/) no final e o comando <br> não possui? Porque o comando <img> é um comando que pode conter atributos, como src e alt, e a barra (/) indica que o comando está sendo fechado. Já o comando <br> é um comando simples que não possui atributos, então não é necessário usar a barra (/) para fechá-lo.

Estilização

Podemos estilizar as tags HTML adicionando na tag de entrada o atributo style e definindo propriedades. Por exemplo:

<h1 style="color: blue;">Título em azul</h1>

Este código estiliza o título com a cor azul.

Mas isso não é recomendado, pois mistura a estrutura do documento com a apresentação visual, o que pode dificultar a manutenção do código. Em vez disso, é melhor usar arquivos CSS externos para separar a estrutura do documento da apresentação visual. Desse modo, vale muito a pena estudarmos um pouco sobre a estrutura do CSS.

O que é CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de um documento HTML. Ele permite que os desenvolvedores controlem a aparência visual das páginas web, como cores, fontes, espaçamento, layout e muito mais.

O CSS é composto por regras de estilo que consistem em um seletor e um bloco de declarações. O seletor é usado para selecionar os elementos HTML que serão estilizados, enquanto o bloco de declarações contém as propriedades e valores que definem a aparência dos elementos selecionados. Por exemplo:

h1 {
    color: blue;
    font-size: 24px;
}

Este código CSS estiliza todos os elementos <h1> com a cor azul e um tamanho de fonte de 24 pixels.

O CSS pode ser aplicado a um documento HTML de várias maneiras, incluindo:

  • Estilos embutidos: Usando o atributo style diretamente nas tags HTML, como mostrado anteriormente.
  • Estilos internos: Colocando as regras de estilo dentro de uma tag <style> no cabeçalho do documento HTML.
  • Estilos externos: Criando um arquivo CSS separado e vinculando-o ao documento HTML usando a tag <link> no cabeçalho do documento HTML.

A prática recomendada é usar estilos externos, pois isso permite uma melhor organização do código e facilita a manutenção e reutilização dos estilos em várias páginas do site.

A utilização de estilos externos também melhora o desempenho da página, pois os arquivos CSS podem ser armazenados em cache pelo navegador, reduzindo o tempo de carregamento das páginas subsequentes.

Temos várias formas de aplicar estilos em um elemento HTML usando CSS. A seguir, estão algumas das formas mais comuns:

  • Seletores de tipo: Estilizam todos os elementos de um determinado tipo. Por exemplo, h1 { color: blue; } estiliza todos os elementos <h1> com a cor azul.
  • Seletores de classe: Estilizam elementos com uma classe específica. Por exemplo, .minha-classe { font-size: 18px; } estiliza todos os elementos com a classe "minha-classe" com um tamanho de fonte de 18 pixels.
  • Seletores de ID: Estilizam um elemento com um ID específico. Por exemplo, #meu-id { background-color: yellow; } estiliza o elemento com o ID "meu-id" com um fundo amarelo.

Uma classe é um conjunto de estilos que podem ser aplicados a vários elementos HTML. Isso permite reutilizar estilos em diferentes partes do documento, tornando o código mais organizado e fácil de manter.

Um ID é um identificador único para um elemento HTML. Ele deve ser usado apenas uma vez em um documento e é útil para estilizar um elemento específico ou para criar links internos dentro da página.

Opções de Estilização Dentro do CSS

Dentro do CSS, existem diversas opções de estilização que podem ser aplicadas aos elementos HTML. Algumas das opções mais comuns incluem:

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo de um elemento.
  • font-size: Define o tamanho da fonte.
  • font-family: Define a família de fontes a ser utilizada.
  • font-style: Define o estilo da fonte (por exemplo, normal, italic, oblique).
  • margin: Define a margem externa de um elemento.
  • padding: Define o preenchimento interno de um elemento.
  • border: Define a borda de um elemento.
  • border-radius: Define o raio da borda, permitindo criar bordas arredondadas.
  • width: Define a largura de um elemento.
  • height: Define a altura de um elemento.
  • display: Define como um elemento é exibido (por exemplo, block, inline, flex, grid).
  • background-color: Define a cor de fundo de um elemento.
  • text-align: Define o alinhamento do texto dentro de um elemento (por exemplo, left, right, center, justify).

Dentro dessas opções citadas acima podemos também utilizar valores específicos para cada propriedade, como cores em formato hexadecimal, RGB ou palavras-chave, tamanhos em pixels, porcentagens ou unidades relativas, entre outros. A combinação dessas propriedades e valores permite criar designs personalizados e responsivos para as páginas web.

Por exemplo, para definir a cor de fundo de um elemento, podemos usar o valor #FF0000 para vermelho, ou rgb(255, 0, 0) para o mesmo resultado em formato RGB. Para definir o tamanho da fonte, podemos usar valores como 16px para um tamanho fixo ou 1.5em para um tamanho relativo ao elemento pai.

Para estilizar o Margin e o Padding, podemos usar valores específicos em pixels ou porcentagens. Por exemplo:

  • margin: 10px;: Define uma margem de 10 pixels em todos os lados do elemento.
  • padding: 15px;: Define um preenchimento de 15 pixels em todos os lados do elemento.

Também podemos usar valores específicos para cada lado do elemento, como:

  • margin-top: 10px;: Define uma margem de 10 pixels na parte superior do elemento.
  • margin-right: 20px;: Define uma margem de 20 pixels na parte direita do elemento.
  • margin-bottom: 10px;: Define uma margem de 10 pixels na parte inferior do elemento.
  • margin-left: 20px;: Define uma margem de 20 pixels na parte esquerda do elemento.
  • padding-top: 15px;: Define um preenchimento de 15 pixels na parte superior do elemento.
  • padding-right: 15px;: Define um preenchimento de 15 pixels na parte direita do elemento.
  • padding-bottom: 15px;: Define um preenchimento de 15 pixels na parte inferior do elemento.
  • padding-left: 15px;: Define um preenchimento de 15 pixels na parte esquerda do elemento.

Essas opções de estilização permitem criar layouts personalizados e responsivos para as páginas web, garantindo uma melhor experiência para os usuários em diferentes dispositivos e tamanhos de tela. Também pode ser feito colocando esses valores em uma única linha, como margin: 10px 20px 10px 20px; para definir margens específicas em cada lado do elemento. Lembrando de respeitar o padrão de ordem: top, right, bottom, left.
