Páginas Web Interativas

O que é JavaScript?

JavaScript é uma linguagem de programação que permite criar páginas web interativas. Com JavaScript, podemos adicionar funcionalidades dinâmicas às nossas páginas, como animações, validação de formulários, manipulação de eventos e muito mais.

Um documento JavaScript é da mesma forma que criamos um arquivo CSS para estilizar nossas páginas HTML, podemos criar um arquivo JavaScript para adicionar interatividade.

Para adicionar JavaScript a uma página HTML, podemos usar a tag <script> dentro do arquivo HTML ou criar um arquivo separado com a extensão .js e vinculá-lo ao HTML.

Tipos

Em JavaScript, existem vários tipos de dados, incluindo:

  • String: representa texto, delimitado por aspas simples ou duplas. Exemplo: "Olá, mundo!"
  • Número: representa valores numéricos, como inteiros e decimais. Exemplo: 42 ou 3.14
  • Booleano: representa valores lógicos, que podem ser true ou false.
  • Objeto: representa uma coleção de propriedades e métodos. Exemplo: { nome: "Felipe", idade: 30 }

String

Em JavaScript, uma string é uma sequência de caracteres usada para representar texto. As strings são delimitadas por aspas simples (' ') ou aspas duplas (" "). Por exemplo:

"Olá, mundo!";
'Felipe';

Neste exemplo, temos duas strings: "Olá, mundo!" e 'Felipe'. As strings podem conter letras, números, espaços e caracteres especiais.

Variável

Um valor é um dado que pode ser armazenado em uma variável.

Em JavaScript, uma variável é um contêiner que armazena um valor. Podemos criar uma variável usando a palavra-chave let ou const. Por exemplo:

let nome = "Felipe";
const idade = 30;

Neste exemplo, criamos uma variável chamada nome que armazena a string "Felipe" e uma variável constante chamada idade que armazena o número 30.

As variáveis podem ser usadas para armazenar diferentes tipos de dados, como números, strings, booleanos, objetos e muito mais.

A palavra-chave const é usada para declarar variáveis cujo valor não pode ser reatribuído após a inicialização. Ou seja, uma vez atribuído um valor a uma variável const, ele não pode ser alterado.

Por exemplo:

const pi = 3.14;
pi = 3.14159; // Isso causará um erro, pois pi é uma constante e não pode ser reatribuída.

Já a palavra-chave let é usada para declarar variáveis que podem ser reatribuídas. Ou seja, o valor de uma variável declarada com let pode ser alterado ao longo do tempo.

Por exemplo:

let contador = 0;
contador = contador + 1; // Isso é permitido, pois contador é uma variável declarada com let e pode ser reatribuída.

Em resumo, use const para variáveis cujo valor não deve mudar e let para variáveis que podem ser reatribuídas ao longo do tempo.

Para declarar números e booleanos não precisamos usar as aspas, pois eles são representados diretamente como valores numéricos.

Por exemplo:

let idade = 30; // número
let ativo = true; // booleano

Neste exemplo, a variável idade armazena o número 30 e a variável ativo armazena o valor booleano true.

Operadores Aritméticos, Atribuição e Comparação

Em JavaScript, os operadores são símbolos ou palavras-chave que realizam operações em valores ou variáveis. Existem vários tipos de operadores, incluindo:

  • Operadores aritméticos: usados para realizar operações matemáticas, como adição (+), subtração (-), multiplicação (*), divisão (/) e módulo (%).
  • Operadores de comparação: usados para comparar valores, como igual a (==), estritamente igual a (===), diferente de (!=), estritamente diferente de (!==), maior que (>), menor que (<), maior ou igual a (>=) e menor ou igual a (<=).
  • Operador de Atribuição: usado para atribuir valores a variáveis, como igual (=).
  • Operadores Lógicos: usados para combinar expressões lógicas, como AND (&&), OR (||) ou NOT (!).

Exemplo de uso dos operadores:

let a = 10;
let b = 5;
let soma = a + b; // 15
let subtracao = a - b; // 5
let multiplicacao = a * b; // 50
let divisao = a / b; // 2
let modulo = a % b; // 0

let igual = a == b; // false
let estritamenteIgual = a === b; // false
let diferente = a != b; // true
let estritamenteDiferente = a !== b; // true
let maior = a > b; // true
let menor = a < b; // false
let maiorOuIgual = a >= b; // true
let menorOuIgual = a <= b; // false

let atribuicao = a = b; // 5

Neste exemplo, realizamos várias operações aritméticas, de comparação e de atribuição usando os operadores mencionados.

Também podemos somar strings:

let saudacao = "Olá, ";
let nome = "Felipe";
let mensagem = saudacao + nome; // "Olá, Felipe"

Neste exemplo, usamos o operador de adição (+) para concatenar as strings "Olá, " e "Felipe", resultando na mensagem "Olá, Felipe".

Além disso, podemos usar o operador de adição para somar números e strings juntos. Quando um dos operandos é uma string, o JavaScript converte o outro operando em uma string e realiza a concatenação. Por exemplo:

let idade = 30;
let mensagemIdade = "Tenho " + idade + " anos."; // "Tenho 30 anos."

Neste exemplo, o número 30 é convertido em uma string e concatenado com as outras strings para formar a mensagem "Tenho 30 anos."

Essa conversão automática de tipos é conhecida como coerção de tipo em JavaScript.

É importante estar ciente dessa coerção de tipo ao usar operadores, especialmente o operador de adição, para evitar resultados inesperados.

E somamos também, strings com booleanos:

let ativo = true;
let mensagemAtivo = "O usuário está " + ativo; // "O usuário está true"

Neste exemplo, o valor booleano true é convertido em uma string e concatenado com a outra string para formar a mensagem "O usuário está true".

O operador de atribuição (=) é usado para atribuir valores a variáveis. Por exemplo, let idade = 30;

Os operadores lógicos são usados para combinar expressões lógicas, como AND (&&), OR (||) ou NOT (!). Por exemplo, 

let resultado = true && false;, 
let resultadoOu = true || false;,
let resultadoNot = !true;

Usando o parêntese, podemos agrupar expressões lógicas para controlar a ordem de avaliação e negar um conjunto de expressões. Por exemplo, 

let resultado = (true || false) && !false;
let outroResultado = !(true && false) || true;

Console.log()

A função console.log() é usada para imprimir mensagens no console do navegador. Ela é útil para depuração e para exibir informações durante o desenvolvimento de uma aplicação JavaScript.

Exemplo de uso do console.log():

let nome = "Felipe";
console.log("Olá, " + nome + "!"); // Imprime "Olá, Felipe!"

Neste exemplo, usamos console.log() para imprimir uma mensagem de saudação que inclui o valor da variável nome.

O console.log é usado apenas para ver o resultado no console do navegador e não afeta o conteúdo da página.

Condicionais

As estruturas condicionais permitem executar diferentes blocos de código com base em uma condição. A estrutura mais comum é o if. Por exemplo:

let idade = 18;
if (idade >= 18) {
    console.log("Você é maior de idade.");
}
 
Se a idade for maior que 18, a mensagem "Você é maior de idade." será exibida no console. Caso contrário, a mensagem não será exibida.

 
A estrutura da condicional é composta por uma expressão condicional entre parênteses e um bloco de código entre chaves. Considerando que um bloco de código é um conjunto de instruções que serão executadas se a condição for verdadeira.


Também podemos usar a estrutura else para executar um bloco de código alternativo quando a condição do if for falsa. Por exemplo:


let idade = 16;
if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}


Neste exemplo, se a idade for menor que 18, a mensagem "Você é menor de idade." será exibida no console.


Logo, a condicional else é ativada quando a condição do if é falsa.


Podemos também usar a estrutura else if para testar múltiplas condições. Por exemplo:


let idade = 20;
if (idade < 18) {
    console.log("Você é menor de idade.");
} else if (idade < 65) {
    console.log("Você é adulto.");
} else {
    console.log("Você é idoso.");
}


Neste exemplo, se a idade for menor que 18, a mensagem "Você é menor de idade." será exibida. Se a idade for entre 18 e 64, a mensagem "Você é adulto." será exibida. Caso contrário, a mensagem "Você é idoso." será exibida.


Podemos utilizar a estrutura else if múltiplas vezes para testar várias condições em sequência. Por exemplo,


let nota = 85;
if (nota >= 90) {
    console.log("Excelente!");
} else if (nota >= 80) {
    console.log("Muito bom!");
} else if (nota >= 70) {
    console.log("Bom!");
} else if (nota >= 60) {
    console.log("Suficiente!");
} else {
    console.log("Insuficiente!");
}


Neste exemplo, a nota é avaliada em várias faixas para determinar a mensagem apropriada a ser exibida no console.
